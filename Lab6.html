We now need to generate the signal which will also be counting particle events. This means that our signal will also be Poisson distributed. We want the signal to be in the $8\sigma$ to $30\sigma$ range. We will pick 250 as the mean value for this Poisson signal injection, which falls within that range.

In [6]:
mixed = np.zeros(r.size)
for i in range(mixed.size):
    if i < signal.size:
        mixed[i] = np.add(r[i], signal[i])
    else:
        mixed[i] = r[i]
plt.hist(mixed,bins = 450)
plt.title('Data After Signal Injection')
plt.xlabel('Measurement')
plt.ylabel('Counts')
plt.show()

As we can see now, the mean has moved to 350. From the prior graph we note that mean was 250, which means that it has moved 100 to the right. This number is also the mean on our Poisson distributed background. This is because now we have both the background and the signal together, which means that both events are happening at the same time now. Because we are now taking both into account the average measurement is now going to be 100 + 250 which gives us the expected 350.
The shape has been retained because the sum of two Poisson distributions is still a Poisson. The measurement we are getting here is that of both- the background and the signal. Which follows that the overall measurement is greater than a measurement of just the background.

3C¶

Now we reverse, so we want to know the what the original injected signal was from the observed one. We saw from previous questions that they only were offset by the mean, so we can subtrack the background and be left with the signal. For this example we will use mean of 400.

new_mean = 400
bg_mean = 100
new_sample = stats.poisson.rvs(new_mean, size=10000)
al = np.array([])
for i in new_sample:
    al = np.append(al, i - bg_mean)
    
plt.hist([new_sample, al], bins = 250, color=['lightblue', 'orange'], label=['Total observed counts', 'Injected signal counts'])

We can confirm again that these distributions look like previous distributions. The injected signal distribution is independent of the background, it is made from the signal alone and without contribution from the background.

3D 3E¶

We can start by normalizing to obtain a pdf. We can then integrate it from the mean to the 1 sigma probability on both sides. In our case the center is 300.

In [12]:
samp1 = stats.norm.cdf(1)
N = a1.size
r = 1
p = 0
while p < samp1:
    p = 0
    for i in al:
        if (i <= r + 300) and (i >= -r + 300):
            p += (1 / N)
    r += 1
print('We integrated up to a distance of {} units from the mean.'.format(r))

We integrated up to a distance of 29 units from the mean.

The range above is the range of our 1 sigma uncertainty in the signal strength. Since we already knew the distribution was symmetric, we could integrate on both sides starting from the center.

4C¶

We normalize the signal to make it into a pdf and integrate from negative infinity up until 95% of the area has been integrated. The statistical question is: if I observe this candidate signal (and it is too weak to claim a detection), then the true signal would be less than X 95% of the time.

In [16]:
M = a1.size
p = 0
r = 1
conf = 0.95
while p < conf:
    p = 0
    for i in a1:
        if i <= r:
            p += (1 / M)
    r += 1

print('We integrated up to a value of {} units above zero.'.format(r))

We integrated up to a value of 34 units above zero.

So, 95% of the time, the true signal strength will be less than r = 34 units above zero.
